# 设置zerotier下的共享文件夹
选择一个想要共享的文件夹》右键》属性》共享》下拉箭头选择everyone》共享》完成

高级共享》权限》完全控制 更改 读取

添加》高级》立即查找》everyone ，everyone就添加进去了
